# InGameRecipeEditor

一款开源 Minecraft Forge 模组，用于通过 JEI 界面在游戏运行时动态管理和生成合成配方。

该模组在 JEI 配方查看器中提供了直观的按钮，无需重启游戏即可禁用、编辑、复制和创建配方。它包含用于配方编辑的工作区系统、自定义标签管理，以及基于模式的、可扩展的配方处理器，以兼容其他模组。

本项目正在积极开发中，采用 Apache-2.0 许可证发布。

## 功能特性

- 在 JEI 界面中集成了配方管理按钮（禁用/启用、编辑、复制、添加）
- 工作区编辑器，支持通过原料选择器创建和修改配方
- 自定义标签的创建和管理系统
- 可扩展的配方模式系统，支持原版及模组配方（如 Mekanism 等）
- 为中文用户提供基于拼音的搜索支持
- 兼容多个流行模组

## 配方编辑系统

基于模式（Schema）的架构，配合工作区编辑器及网络同步。

**已完成**：启用/禁用、删除、工作区 UI、化学渲染、配方模式注册。

**开发中**：工作区编辑回调、提交机制、JSON 生成、属性编辑。
